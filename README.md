# Improved-PreCache-System
Garry's Mod / Inj3 - Improved Precache System

- Optimized code, execute only once when the server has initialized entities (no impact on performance.)
- Compatible with all gamemodes.
- Configuration (lua/autorun/server).
- Define delay between each caching phase.
- Support for a large majority of vehicles such as TDM, and all those in the vehicle list.
- Caching your models and sounds.
- Blacklist vehicles that should not be included in the cache.
- Detailed statistics during caching in your console.
